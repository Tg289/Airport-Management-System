# Airport Management System

## Description
A desktop-based Java application to manage airport operations.  
Supports managing flights and passengers with a GUI interface and MySQL backend.

## Features
- Add and view flights
- Add and view passengers
- GUI interface using Java Swing
- MySQL database for storage

## Tech Stack
- Java (Swing)
- MySQL
- JDBC

## Setup Instructions
1. Clone the repository
2. Add MySQL connector JAR to `lib/`
3. Run `database.sql` in MySQL to create database and tables
4. Update `config.properties` with your DB credentials
5. Compile and run `Main.java`
